# Docker 地址<br>
> **获取docker镜像**<br>
> x86-64 架构<br>
`docker pull imkenf/xueqg`
> arm64 架构<br>
`docker pull imkenf/xueqg-arm64v8`
> arm32 架构<br>
`docker pull imkenf/xueqg-arm32v7`

> **相关说明请查看：**<br>
